== だけではなく(て) 的用法

=== 基本結構

**だけではなく(て)** 是一個常用的日語句型，用來表示「不僅～而且～」、「不只是～也～」。

[cols="2,2,3", options="header"]
|===
|句型 |意思 |例句

|A だけではなく B も
|不僅 A，B 也
|日本語だけではなく、英語も話せる

|A だけでなく B も
|不僅 A，B 也（較正式）
|学生だけでなく、先生も参加した

|A だけじゃなく(て) B も
|不僅 A，B 也（口語）
|美味しいだけじゃなくて、安い

|===

=== 完整形式變體

**從正式到口語的各種形式：**

[cols="2,2,2", options="header"]
|===
|形式 |正式度 |使用場合

|だけではなく
|最正式
|書面語、演講、新聞

|だけでなく
|正式
|書面語、一般文章

|だけではなくて
|普通
|口語、日常對話

|だけでなくて
|普通
|口語、日常對話

|だけじゃなく
|口語
|日常對話、朋友間

|だけじゃなくて
|口語
|日常對話、朋友間

|===

== 基本用法

=== 結構說明

**基本句型：**

```
A だけではなく(て)、B も～
Not only A, but also B
```

**功能：** 列舉多個項目，強調不僅有 A，還有 B

=== 接續方式

**名詞 + だけではなく(て)**

* 日本語**だけではなく**、英語も話せる
** 不僅日語，英語也會說

**動詞辭書形/た形 + だけではなく(て)**

* 読む**だけではなく**、書くこともできる
** 不僅能讀，也能寫

**い形容詞 + だけではなく(て)**

* 美味しい**だけではなく**、栄養もある
** 不僅好吃，也有營養

**な形容詞 + だけではなく(て)**

* 便利**なだけではなく**、安全でもある
** 不僅方便，也很安全

== 詳細用法

=== 1. 名詞 + だけではなく

**例句：**

* 日本語だけではなく、中国語も勉強している
** 不僅學日語，也在學中文

* 東京だけではなく、大阪にも行った
** 不僅去了東京，也去了大阪

* 学生だけではなく、先生も参加した
** 不僅學生，老師也參加了

* 子供だけではなく、大人も楽しめる
** 不僅小孩，大人也能享受

* 日本だけではなく、世界中で有名だ
** 不僅在日本，在全世界都很有名

**結構：**
```
名詞 + だけではなく(て)、[名詞] + も
```

=== 2. 動詞 + だけではなく

**辭書形：**

* 読むだけではなく、書くこともできる
** 不僅能讀，也能寫

* 見るだけではなく、実際にやってみよう
** 不僅要看，也要實際做做看

* 勉強するだけではなく、復習も大切だ
** 不僅學習，複習也很重要

**た形：**

* 買っただけではなく、使ってみた
** 不僅買了，也試用了

* 聞いただけではなく、自分でも調べた
** 不僅聽了，也自己調查了

* 見ただけではなく、写真も撮った
** 不僅看了，也拍了照片

**結構：**
```
動詞辭書形/た形 + だけではなく(て)、[動詞] + も
```

=== 3. い形容詞 + だけではなく

**例句：**

* 美味しいだけではなく、栄養もある
** 不僅好吃，也有營養

* 安いだけではなく、品質も良い
** 不僅便宜，品質也好

* 大きいだけではなく、丈夫でもある
** 不僅大，也很耐用

* 難しいだけではなく、時間もかかる
** 不僅難，也很花時間

* 楽しいだけではなく、勉強にもなる
** 不僅有趣，也能學到東西

**結構：**
```
い形容詞 + だけではなく(て)、[形容詞/名詞] + も
```

=== 4. な形容詞 + だけではなく

**例句：**

* 便利なだけではなく、安全でもある
** 不僅方便，也很安全

* きれいなだけではなく、性格も良い
** 不僅漂亮，性格也好

* 有名なだけではなく、実力もある
** 不僅有名，也有實力

* 静かなだけではなく、景色も素晴らしい
** 不僅安靜，景色也很美

* 簡単なだけではなく、効果的でもある
** 不僅簡單，也很有效

**結構：**
```
な形容詞 + なだけではなく(て)、[形容詞] + も
```

== 口語變體

=== だけじゃなく(て)（口語形式）

**特徵：** では → じゃ（口語縮約）

**例句：**

* 日本語だけじゃなく、英語も話せる
** 不僅日語，英語也會說

* 美味しいだけじゃなくて、安いよ
** 不僅好吃，也便宜喔

* 勉強だけじゃなく、遊びも大事だよ
** 不僅學習，玩樂也很重要喔

* 見たダケじゃなくて、実際に行った
** 不僅看了，也實際去了

* 難しいだけじゃなく、面倒くさい
** 不僅難，也很麻煩

**使用場合：**

* ✅ 朋友之間的對話
* ✅ 日常輕鬆的交流
* ✅ 社交媒體、即時通訊
* ❌ 正式場合、商務會議
* ❌ 正式文件、報告

=== 省略「て」的形式

**だけではなく** vs **だけではなくて**

兩者意思相同，但用法略有不同：

**だけではなく（不加て）：**

* 更簡潔
* 常用於書面語
* 例：日本だけではなく、世界中で人気だ

**だけではなくて（加て）：**

* 語氣較柔和
* 常用於口語
* 例：美味しいだけではなくて、栄養もあるよ

== 相關句型比較

=== 1. だけではなく vs ばかりでなく

**だけではなく：**

* 更常用、普通
* 口語和書面都可以
* 例：学生だけではなく、先生も

**ばかりでなく：**

* 較正式、文語
* 強調程度更高
* 例：日本ばかりでなく、世界中で有名だ

**對比例句：**

* 日本語だけではなく、英語も話せる（普通）
* 日本語ばかりでなく、英語も話せる（較正式）

=== 2. だけではなく vs のみならず

**だけではなく：**

* 日常用語
* 口語化
* 例：東京だけではなく、大阪も

**のみならず：**

* 非常正式
* 書面語、演講
* 例：東京のみならず、大阪も

**對比例句：**

* 学生だけではなく、教師も参加した（普通）
* 学生のみならず、教師も参加した（正式）

=== 3. だけではなく vs だけでなく

**だけではなく：**

* 完整形式
* 較正式

**だけでなく：**

* 省略 は
* 同樣正式，更簡潔

**對比：**

* 日本語だけ**では**なく、英語も話せる
* 日本語だけなく、英語も話せる

兩者**意思完全相同**，だけでなく 較簡潔常用。

=== 4. だけではなく vs も～も

**だけではなく：**

* 強調「不僅」的語氣
* 有遞進的感覺
* 例：美味しいだけではなく、栄養もある

**も～も：**

* 單純並列
* 沒有強調遞進
* 例：美味しいし、栄養もある

**語感差異：**

* だけではなく：「不僅 A，**而且還有** B」（強調）
* も～も / し：「A，**也** B」（平等並列）

== 實際會話範例

=== 場景1：介紹自己的技能

**正式場合（面試）：**

A: どんなスキルをお持ちですか？
B: 日本語だけでなく、英語と中国語も話せます

* A: 您有什麼技能？
* B: 不僅日語，我也會說英語和中文

**日常對話：**

A: 何語話せるの？
B: 日本語だけじゃなくて、英語と中国語も話せるよ

* A: 你會說什麼語言？
* B: 不僅日語，英語和中文也會說喔

=== 場景2：推薦餐廳

**正式：**

このレストランは料理だけではなく、雰囲気も素晴らしいです

* 這家餐廳不僅料理，氣氛也很棒

**口語：**

このレストラン、料理だけじゃなくて、雰囲気もいいよ

* 這家餐廳不僅料理，氣氛也不錯喔

=== 場景3：描述商品

**書面語（廣告）：**

この製品は便利なだけでなく、環境にも優しい

* 本產品不僅方便，也環保

**口語（店員介紹）：**

これ、便利なだけじゃなくて、環境にも優しいんですよ

* 這個不僅方便，對環境也很友善喔

=== 場景4：討論學習

**正式：**

語学の勉強は聞くだけではなく、話す練習も重要です

* 語言學習不僅要聽，說的練習也很重要

**口語：**

語学って、聞くだけじゃなくて、話す練習も大事だよね

* 語言學習不僅要聽，說的練習也很重要呢

== 常見搭配模式

=== 模式1：並列兩個名詞

```
名詞A + だけではなく(て)、名詞B + も
```

* 学生だけではなく、先生も
* 日本だけではなく、中国も
* 男性だけではなく、女性も

=== 模式2：並列兩個動作

```
動詞A + だけではなく(て)、動詞B + も
```

* 読むだけではなく、書くことも
* 見るだけではなく、聞くことも
* 勉強するだけではなく、復習することも

=== 模式3：並列兩個性質

```
形容詞A + だけではなく(て)、形容詞B + も
```

* 美味しいだけではなく、安くも
* 大きいだけではなく、丈夫でも
* 便利なだけではなく、安全でも

=== 模式4：三個以上的並列

```
A だけではなく、B も、C も
```

* 日本語だけではなく、英語も中国語も話せる
** 不僅日語，英語和中文也會說

* 東京だけではなく、大阪も京都も行った
** 不僅去了東京，大阪和京都也去了

== 省略與變化

=== 省略「も」

有時候可以省略第二個項目的「も」：

**標準形式：**
* 美味しいだけではなく、栄養**も**ある

**省略も：**
* 美味しいだけではなく、栄養がある

兩者都正確，但帶「も」更強調並列關係。

=== 使用其他助詞

第二個項目不一定要用「も」，可以根據需要使用其他助詞：

* 日本語だけではなく、英語**で**話した
** 不僅用日語，也用英語說了

* 学生だけではなく、先生**に**も聞いた
** 不僅問了學生，也問了老師

* 東京だけではなく、大阪**へ**も行く
** 不僅去東京，也去大阪

=== 否定形式

可以在後面接否定：

* 日本語だけではなく、英語も話せ**ない**
** 不僅日語，連英語也不會說（都不會）

但這種用法較少見，更常用的是：

* 日本語だけでなく、英語**さえ**話せない
** 不僅日語，連英語也不會說

== 特殊用法

=== だけではなくて的獨立使用

有時「だけではなくて」可以單獨使用，後面不一定要接「も」：

**例句：**

A: これは美味しいね
B: 美味しいだけじゃなくて、栄養もあるんだよ

* A: 這個很好吃呢
* B: 不僅好吃，也很有營養喔

**強調用法：**

* ただ美味しいだけではなくて！栄養もあるんです！
** 不僅僅是好吃而已！也很有營養！

=== 與「も」分開使用

有時「だけではなく」和「も」會被較長的內容分開：

* 日本語だけではなく、私が住んでいたアメリカで使われている英語**も**話せる
** 不僅日語，我住過的美國使用的英語也會說

== 練習題

=== 練習1：選擇適當的形式

**請選擇最適合的形式：**

1. 商務會議中說：「この製品は便利____、価格も安いです」
   * a) だけじゃなくて
   * b) だけではなく
   * **答案：b) だけではなく**（正式場合）

2. 朋友聊天：「美味しい____、安いよ」
   * a) だけじゃなくて
   * b) のみならず
   * **答案：a) だけじゃなくて**（口語）

3. 學術論文：「日本____、アジア全体で見られる現象である」
   * a) だけじゃなく
   * b) のみならず
   * **答案：b) のみならず**（學術文體）

=== 練習2：完成句子

**請完成以下句子：**

1. 日本語だけではなく、________
   * 參考答案：英語も話せる／中国語も勉強している

2. 美味しいだけではなく、________
   * 參考答案：栄養もある／安い

3. 学生だけではなく、________
   * 參考答案：先生も参加した／親も来た

4. 読むだけではなく、________
   * 參考答案：書くこともできる／話すこともできる

=== 練習3：改寫句子

**將以下句子改為口語形式：**

1. 正式：日本語だけではなく、英語も話せます
   * **口語：** 日本語だけじゃなくて、英語も話せるよ

2. 正式：東京だけでなく、大阪にも行きました
   * **口語：** 東京だけじゃなく、大阪にも行った

3. 正式：便利なだけではなく、安全でもある
   * **口語：** 便利なだけじゃなくて、安全でもあるよ

=== 練習4：翻譯練習

**中文 → 日文**

1. 不僅學生，老師也參加了
   * **答案：** 学生だけではなく、先生も参加した

2. 不僅好吃，也便宜
   * **答案：** 美味しいだけではなく、安い

3. 不僅能讀，也能寫
   * **答案：** 読むだけではなく、書くこともできる

4. 不僅在日本，在世界各地都很有名（口語）
   * **答案：** 日本だけじゃなくて、世界中で有名だ

5. 不僅方便，也很安全（正式）
   * **答案：** 便利なだけでなく、安全でもある

=== 練習5：錯誤改正

**找出並改正以下句子的錯誤：**

1. ❌ 日本語だけではなく、英語が話せる
   * ✅ 日本語だけではなく、英語**も**話せる
   * **說明：** 需要加「も」表示「也」

2. ❌ 美味しいなだけではなく、安い
   * ✅ 美味しい（な去掉）だけではなく、安い
   * **說明：** い形容詞不需要「な」

3. ❌ 便利だけではなく、安全でもある
   * ✅ 便利**な**だけではなく、安全でもある
   * **說明：** な形容詞需要加「な」

== 常見錯誤

=== 錯誤1：忘記加「も」

❌ **錯誤：** 日本語だけではなく、英語が話せる
✅ **正確：** 日本語だけではなく、英語**も**話せる

**說明：** 「だけではなく」通常需要配合「も」來表示「也」

=== 錯誤2：な形容詞接續錯誤

❌ **錯誤：** 便利だけではなく、安全だ
✅ **正確：** 便利**な**だけではなく、安全だ

**說明：** な形容詞需要加「な」再接「だけではなく」

=== 錯誤3：い形容詞加了「な」

❌ **錯誤：** 美味しい**な**だけではなく、安い
✅ **正確：** 美味しいだけではなく、安い

**說明：** い形容詞直接接「だけではなく」，不需要「な」

=== 錯誤4：正式場合使用口語形式

❌ **不當：** 商務會議中說「便利なだけじゃなくて、安いです」
✅ **適當：** 「便利なだけでなく、安価でもあります」

**說明：** 正式場合應使用正式形式

=== 錯誤5：混淆「だけではなく」和「だけでなく」

這兩者都正確，但要注意一致性：

✅ **正確：** だけ**では**なく...～**では**ない
✅ **正確：** だけ**で**なく...～**で**ない

保持文體一致性！

=== 同義表達

**表示「不僅～而且～」的其他方式：**

1. **～し、～し**
   * 美味しいし、安いし、最高だ
   * 好吃，又便宜，真棒

2. **～ばかりか**
   * 雨ばかりか、雪も降った
   * 不僅下雨，連雪也下了

3. **～はもちろん**
   * 日本語はもちろん、英語も話せる
   * 日語當然會，英語也會說

4. **～に加えて**
   * 日本語に加えて、英語も勉強している
   * 除了日語，也在學英語

== 總結

=== 核心句型

**だけではなく(て)、～も**

* 意思：不僅～而且～
* 功能：並列兩個或以上的項目
* 強調：遞進關係，不僅有 A，還有 B

=== 形式變體

**正式 → 口語：**

1. のみならず（最正式）
2. ばかりでなく
3. だけでなく
4. だけではなく
5. だけではなくて
6. だけでなくて
7. だけじゃなく
8. だけじゃなくて（最口語）

=== 接續規則

* **名詞：** 名詞 + だけではなく
* **動詞：** 動詞辭書形/た形 + だけではなく
* **い形容詞：** い形容詞 + だけではなく
* **な形容詞：** な形容詞 + **な** + だけではなく

=== 使用要點

1. **配合「も」使用**
   - 通常第二個項目要加「も」

2. **注意正式度**
   - 正式場合用「だけでなく」或「だけではなく」
   - 口語用「だけじゃなく(て)」

3. **な形容詞要加「な」**
   - 便利**な**だけではなく

4. **可以並列多個項目**
   - A だけではなく、B も、C も

5. **強調遞進關係**
   - 比單純的「も～も」更強調

=== 記憶技巧

**結構拆解：**

* **だけ** = only（只是）
* **では** = で + は（助詞）
* **なく** = ない的連用形（不）
* **て** = 連接助詞

**直譯：** 不只是...而已

**意譯：** 不僅...而且...

**口訣：**
「だけではなく」後面要加「も」，
な形容詞中間插個「な」，
正式口語看場合，
「では」「じゃ」隨你選！

=== 最後提醒

1. **最常用的形式**是「だけではなく」和「だけでなく」
2. **口語對話**中「だけじゃなくて」最自然
3. **正式文書**用「だけでなく」或「ばかりでなく」
4. 記得配合「**も**」使用
5. な形容詞要加「**な**」
6. 可以連接各種詞性
7. 注意保持文體一致性

掌握「だけではなく(て)」這個句型，能讓你的日語表達更加豐富，更有層次感！
